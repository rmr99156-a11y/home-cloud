# Домашнее облако

Лёгкий локальный файловый сервер на Flask, который превращает ваш старый ПК в персональное облако.
Позволяет загружать, скачивать, удалять файлы и папки, искать файлы по имени и делиться публичными ссылками.

## Возможности

- **Авторизация по логину/паролю** — защищённый доступ только для вас
- **Управление файлами** — загрузка, скачивание, удаление, создание папок
- **Поиск** — мгновенный поиск по всем файлам в хранилище
- **Публичные ссылки** — поделитесь файлом с кем угодно (без регистрации)
- **QR-код** — отсканируйте, чтобы быстро открыть облако с телефона
- **Иконки по типам файлов** — собственные SVG-иконки для картинок, видео, музыки, документов, архивов
- **Индикатор свободного места** — наглядный прогресс-бар занятости диска
- **Статистика** — общее количество файлов и размер хранилища
- **Тёмно-зелёная тема** — стильный тёмный интерфейс с зелёными акцентами
- **Анимации** — плавные переходы, появление карточек, hover-эффекты
- **Полная мобильная адаптация** — карточный вид файлов, гамбургер-меню, FAB-кнопка, свайпы
- **Drag & Drop** — перетаскивайте файлы прямо в браузер

## Установка

### Требования
- Python 3.8 или новее
- pip (установщик пакетов Python)

### Шаг 1. Установить зависимости

**Linux / macOS:**
```bash
cd local_cloud
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

**Windows (PowerShell):**
```powershell
cd local_cloud
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

После запуска в терминале появятся адреса:
```
  Локальный доступ:  http://127.0.0.1:8000
  Доступ по сети:    http://192.168.1.X:8000
```

Откройте **Локальный доступ** в браузере на этом ПК.

## Первый вход

По умолчанию:
- **Логин:** `admin`
- **Пароль:** `admin`

⚠️ **ОБЯЗАТЕЛЬНО смените пароль** в разделе «Настройки» после первого входа!

## Доступ с других устройств в домашней сети

По умолчанию сервер слушает только `127.0.0.1:8000` (доступ с этого же ПК).
Чтобы открыть доступ с телефона/других ПК в домашней Wi-Fi сети:

1. Откройте `app.py` в текстовом редакторе
2. Найдите строку `HOST = "127.0.0.1"` (примерно строка 30)
3. Замените на `HOST = "0.0.0.0"`
4. Перезапустите сервер
5. Откройте с телефона URL `http://<IP-ВАШЕГО-ПК>:8000`
   (IP показывается в терминале при запуске)

Для доступа с телефона отсканируйте **QR-код** в левом нижнем углу сайдбара.

На телефоне и планшете интерфейс автоматически перестраивается:
- **Гамбургер-меню** слева сверху — открывает сайдбар

## Где хранятся файлы?

Все загруженные файлы лежат в папке `storage/` рядом с `app.py`.
Если хотите хранить на отдельном диске (например, на старом HDD), отредактируйте в `app.py`:
```python
STORAGE_DIR = Path("/mnt/big_hdd/my_cloud_storage")
```

## Безопасность

- Пароль хранится в виде хэша (Werkzeug, не в открытом виде)
- Сессия защищена случайным секретным ключом
- Защита от path-traversal (нельзя выйти за пределы папки `storage/`)
- Имена файлов фильтруются (secure_filename)
- Лимит размера файла — 1 ГБ (можно изменить в `MAX_FILE_SIZE`)

⚠️ **Не публикуйте сервер в интернете без HTTPS и надёжного пароля!**
Для внешнего доступа рекомендуется использовать reverse-proxy (nginx) с TLS.

## Запуск как служба (опционально)

**Linux — systemd:**
Создайте файл `/etc/systemd/system/home-cloud.service`:
```ini
[Unit]
Description=Home Cloud Server
After=network.target

[Service]
Type=simple
User=YOUR_USERNAME
WorkingDirectory=/path/to/local_cloud
ExecStart=/path/to/local_cloud/venv/bin/python /path/to/local_cloud/app.py
Restart=on-failure

[Install]
WantedBy=multi-user.target
```

Затем:
```bash
sudo systemctl daemon-reload
sudo systemctl enable home-cloud
sudo systemctl start home-cloud
```

**Windows — запуск при старте:**
Создайте `.bat` файл в `shell:startup`:
```bat
cd C:\path\to\local_cloud
venv\Scripts\python.exe app.py
```

## Частые вопросы

**Q: Можно ли загружать файлы больше 1 ГБ?**
A: Да, измените `MAX_FILE_SIZE` в `app.py`. Но учтите, что большие файлы могут требовать много RAM при загрузке.

**Q: Как сделать бэкап?**
A: Просто скопируйте папку `storage/` и файл `config.json` на внешний диск.

**Q: Можно ли использовать на Raspberry Pi?**
A: Да, всё работает на ARM. Установите Python 3 и следуйте обычной инструкции.

**Q: Как сбросить пароль?**
A: Удалите файл `config.json` и перезапустите сервер — создастся новая учётка `admin/admin`.
